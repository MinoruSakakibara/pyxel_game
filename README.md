# 剣を避けるゲーム

## 概要
Pythonを使用して初めてゲームを作ってみました。<br>
落下してくる剣をひたすら避けるシンプルなものとなっています。<br>

## 実行環境
Windows 10 / 11

## ダウンロード方法
ドライブから下記ファイルのいずれかをダウンロード。<br>

- pyxel_game_pc > pyxel_game.exe<br>
- pyxel_game_web > pyxel_game.html<br>

※exeファイルはデスクトップ上、htmlファイルはWeb上でプレイが可能です。<br>

## 遊び方
タイトル画面で左クリックを押すとゲームがスタートします。<br>
落下してくる剣に触れるとゲームオーバーです。<br>
ゲームオーバー後は自動でタイトル画面に戻ります。<br>

## ルール
1.剣を避けるたびにスコア加算<br>
2.ゲームオーバー時、その時点でのスコアを表示<br>
3.時間経過で落下する剣の数増加(10秒毎、MAX3段階)<br>
4.時間経過で落下する剣の速度上昇(10秒毎、MAX3段階)<br>

## 操作方法
- 右移動：矢印キー右 or Dキー
- 左移動：矢印キー左 or Aキー
- ジャンプ：スペースキー
